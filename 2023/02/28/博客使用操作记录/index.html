发布文章

进入博客所在目录，右键点击 Git Bash Here ，创建博客

然后可以在source/_posts内查看新创建的 .md文件，在其中编写即可。

在 front-matter 内添加 title，date，tags，description，cover等描述文字。

在博客所在仓库目录下新建 CNAME 文件，输入域名（如果自定义域名的话）：

ssh 连接 Github

解决方法

  • 修改 known_hosts，删除known_hosts.old。（原理参考https://cloud.tencent.com/developer/article/1946906）
  • 删除博客目录下原有的 .deploy_git 和 public 文件。

博客可添加功能

标签外挂

默认：

名称 用法
class 标识，不同的标识有不同的颜色
( default / primary/ success / info / warning / danger )
no-icon 不显示 icon
style 可以覆盖配置中的 style
( simple / modern / flat / disabled )

自定义：

名称 用法
color 颜色 ( default / blue / pink / red / purple / orange / green )
icon 可配置自定义icon（只支持 fontawesome 图标，也可以配置 no-icon ）
style 可以覆盖配置中的 style ( simple / modern / flat / disabled )

Gallery 相册

名称 用法
lazyload 点击按钮加载更多图片，填写 true / false。 默认为 false
rowHeight 图片显示的高度，默认220
limit 每次加载多少张照片，默认为10
